# 🤖 Agente SERPRO Inteligente

## 💡 Proposta do Projeto
Este projeto tem como objetivo **explorar o uso do Amazon Bedrock** em um **problema real**: criar um agente inteligente capaz de responder dúvidas sobre a **Carta de Serviços do SERPRO**.

A proposta é demonstrar, de forma prática, como a tecnologia de modelos de linguagem pode ser aplicada para **facilitar o acesso à informação** em documentos extensos e técnicos, tornando-os **interativos e acessíveis** para os colaboradores.

---

## 🧠 O que o Agente Faz
O agente atua como um **consultor virtual especializado** no conteúdo da Carta de Serviços, permitindo que o usuário faça perguntas em linguagem natural, como:
- “O que é o serviço DaaS do Serpro?”
- “Quem pode contratar o serviço de VPN?”
- “Como funciona o atendimento de suporte técnico?”

As respostas são geradas com base no conteúdo oficial do documento, de forma **resumida, objetiva e contextual**.

---

## 🎯 Objetivo do Projeto
O propósito principal não é apenas criar um chatbot, mas **explorar o potencial do Amazon Bedrock** para:
- Construir um **agente especializado** no domínio dos serviços de TI oferecidos pela empresa;
- Utilizar técnicas de **embeddings e busca semântica** para melhorar a precisão das respostas;
- **Registrar e analisar métricas de uso**, como quantidade de tokens, perguntas e dúvidas mais frequentes;
- Servir como **prova de conceito** de como a IA generativa pode **apoiar os colaboradores** no entendimento do que a empresa pode ofertar a seus clientes.

---

## 🧰 Tecnologias Utilizadas
- **Amazon Bedrock** – processamento de linguagem natural e geração de respostas inteligentes.  
- **Streamlit** – interface simples e interativa para interação com o agente.  
- **SQLite** – armazenamento local das perguntas e métricas de uso.  
- **Python** – base do desenvolvimento e integração entre os componentes.  

---

## 📚 Fonte dos Dados
As informações utilizadas pelo agente são baseadas na **Carta de Serviços do SERPRO**, um documento público que descreve detalhadamente os serviços de tecnologia oferecidos pela empresa.
